About the Lander Coming soon 3 Template

This HTML5 Template is made with Bootstrap v 3.3.6 and it's very easy to modify it as you wish. Just follow the instructions below!

Included files

The main.zip file you have just downloaded contains the following folders:

  • 1. _documentation
  • 2. coming-soon-3 (it contains all the necessary files for your site)

Customizing the Template


LANDER ONE-PAGE COMING SOON 3 TEMPLATE

Inside the coming-soon-3 folder, you'll find the following folders:

  • css (all the needed css files of the template)
  • fonts ( the fonts used inside the template)
  • images ( it contains all the pictures placeholders of the template)
  • js (all the javascript files)
  • lib (all the bootstrap and font-awesome css files )
  • index.html
  • notify-me.php




  • How do i replace the site logo with my own?
    Locate the images folder and replace the logo.png (120 x 28 px) with your own, then upload it to your server. (see the picture below).

  • How do i set my social media addresses?
    Open the index.html file, go to the line 134 and put the link to your social media pages inside there.


  • How do i change the main background picture?
    Inside the main folder, replace the main-bg.jpg image (size 1400x788 or more).
    If you want to change the text above the image as well, open the index.html, start from line 85 and go down in order to replace the text with your own.



 

 

 

  • How do i change the particles animation color above the picture?
    Open the common.js file inside the js folder and go to line 55. Change the dot and the line colors there. (see below)

 

 

 

  • How do i change the main picture opacity?
    Open the index.html file, go to the line 58 and set the opacity as you want. (default is 60)

 

  • How do i set the countdown time of the page?
    Open the common.js file inside the js folder and go to line 214. Change the date there. (see below)



  • How do i change the email form in order to receive subscription from this page?
    Inside the main folder, find the notify-me.php page, open it and replace the email address at line 6 with your own..
